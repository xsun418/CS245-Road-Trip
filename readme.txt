How to compile
javac RoadTrip.java

How to use this program:
You will see the program ask the starting city and ending city,
and you need type the name of city and the abbreviation of state.
like San Francisco CA, Akron OH.

Then it will ask the name the attraction,
you can type the name of attraction.
If you dont want to add the attraction,
you can type -1, and the program will calculate the route
